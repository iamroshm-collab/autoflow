# Attendance & Payroll Management Module

## Overview
A comprehensive attendance and payroll management system with three main features:
1. **Attendance Management** - Daily attendance tracking
2. **Adjustments** - Manage allowances, advances, and incentives
3. **Payroll Generation** - Monthly salary calculation and slip generation

---

## Database Schema

### Models Added to Prisma Schema

#### 1. AttendancePayroll (Enhanced)
- Tracks daily attendance with unique constraint per employee per day
- Fields: `attendance` (P/H/L/A), `salaryAdvance`, `incentive`, `allowance`

#### 2. Adjustment (New)
- Tracks all financial adjustments for employees
- Types: Allowance, Advance, Incentive
- Fields: `adjustmentType`, `amount`, `adjustmentDate`, `remarks`

#### 3. MonthlyPayroll (New)
- Stores generated monthly payroll data
- Auto-calculates: basicSalary, netSalary from attendance and adjustments
- Unique constraint per employee per month/year

---

## API Routes Created

### 1. `/api/attendance-payroll/attendance`
- **GET** - Fetch attendance records for a specific date
  - Query: `?date=2026-02-16`
- **POST** - Create/update single attendance record
  - Body: `{ employeeId, attendanceDate, attendance }`
- **PUT** - Bulk update attendance records
  - Body: `{ attendanceRecords: [] }`

### 2. `/api/attendance-payroll/adjustments`
- **GET** - Fetch adjustments with optional filters
  - Query: `?employeeId=1&month=2&year=2026&search=John`
- **POST** - Create new adjustment
  - Body: `{ employeeId, adjustmentType, amount, adjustmentDate, remarks }`
- **PUT** - Update adjustment
  - Body: `{ adjustmentId, ... }`
- **DELETE** - Delete adjustment
  - Query: `?adjustmentId=1`

### 3. `/api/attendance-payroll/payroll`
- **GET** - Fetch monthly payroll records
  - Query: `?month=2&year=2026&employeeId=1`
- **POST** - Generate monthly payroll for all employees
  - Body: `{ month, year, generatedBy }`
- **DELETE** - Delete payroll record
  - Query: `?payrollId=1`

### 4. `/api/attendance-payroll/employees`
- **GET** - Fetch active employees
  - Query: `?search=John&includeArchived=false`

## Salary Calculation Logic

### Net Salary Formula:
```
Basic Salary = (Present Days + Half Days × 0.5 + Leave Days) × Salary Per Day

Net Salary = Basic Salary + Allowances + Incentives - Advances
```

### Attendance Types:
- **P (Present)** - Full day pay
- **H (Half Day)** - Half day pay (0.5)
- **L (Leave)** - Full day pay (1.0)
- **A (Absent)** - No pay (0.0)

## Validation & Business Rules

### Attendance:
- ✅ Only one attendance record per employee per day
- ✅ Automatic save on change
- ✅ Only allows P, H, L, A values

### Adjustments:
- ✅ Positive amounts only
- ✅ Date required
- ✅ Employee selection required
- ✅ Three types: Allowance, Advance, Incentive

### Payroll:
- ✅ Unique per employee per month/year
- ✅ Auto-calculates from attendance + adjustments
- ✅ Can regenerate/update existing payroll
- ✅ Prevents deletion if employee has dependencies
